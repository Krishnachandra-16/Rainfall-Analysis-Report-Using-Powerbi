## 📌 Project Overview

This project is a **Rainfall Analysis Dashboard** created using **Microsoft Power BI**.  
It focuses on analyzing rainfall patterns across different regions and time periods to understand climate trends and precipitation behavior.

The dashboard transforms raw rainfall data into **interactive and meaningful visual insights**.

---

## 🎯 Objectives

- Analyze historical rainfall data  
- Identify seasonal rainfall trends  
- Compare rainfall across different regions  
- Visualize monthly and yearly rainfall patterns  
- Generate insights for climate and agricultural analysis  

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Excel / CSV Dataset  

---

## 📊 Dashboard Features

- Monthly rainfall trend analysis  
- Region-wise rainfall comparison  
- Yearly rainfall distribution  
- Maximum and minimum rainfall insights  
- Interactive filters (Year, Region, Month)
